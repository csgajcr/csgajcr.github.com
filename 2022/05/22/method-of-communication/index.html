《沟通的方法》读书笔记

发表于 2022-05-22 阅读次数：

前言

精华语录

把对方的信息按照事实、情绪和期待分类处理。

反馈的逻辑：响应情绪-确认事实-明确行动。要让对方明白我们确实听懂了。

不同沟通风格的人的特点

类型 老虎 孔雀 猫头鹰 考拉
特征 控制 取悦 谨慎 温和
害怕 失控 被忽略 混乱 变化
非正常状态 没目标 没表现自己 率先表态 欣然接受变化
有效的沟通方式 目标导向 肯定他 有序 主动维护

肯定反射

例如：
有价值：

  • 您的意见一直对我都很重要
  • 您刚刚说的这些对我太有启发了
    没有价值：
    我没想到，您花了这么多时间跟我聊这事儿

目标感的本质是方案力，具有目标感的沟通，不是抱着自己最初的立场不变，而是要把目标实现。

“我们有一个目标要达成，对此，我有一个方案”

怎么找到真正的目标？判断马上要做的事和最终要做的事

6 个月之后法：通过 6 个月后的愿景来倒推当下要做什么。

找到立即可以执行的最小化行动，是我们把沟通导向行动的起点。

建设性=可执行的最小化行动+可持续的行动阶梯+每个时间节点的即时反馈

破冰=双线卡位+展现关切+营造掌控

和沟通对象破冰，本质上不是展现自己，而是通过给对方营造掌控感，让自己更快的赢得信任，消除彼此的距离感。

找到他与与所在群体之间差异，才是精准的赞美。赞美行为，而不是赞美禀赋。

“每个人毕生都在追求的是被看见”

激励=即时赞美+行为建模+反馈闭环

说服=说话有分量+凿穿心理防线

  1. 有分量 = 强准别
  2. 凿穿心理防线 = 吹风+慢熬

每一个抵抗的情绪背后都有尚未满足的需要

教行动方法，不教价值观，是辅导的基本准则

辅导 = 植入目标 + 发现盲区 + 实战演习

复盘的操作，不应局限于分析哪里做得好，哪里做的不好，还应该更关注思考的过程。

选择与淘汰员工，都比辅导来的更重要。

安慰的前提是能感受他人的安慰

安慰 = 轻度介入 + 提供支持

批评 = 控制环境 + 定义问题 + 刷新动作 + 设定反馈点 + 完成重启

提意见 = 事前征求同意 + 定义双方关系 + 提供具体建议

绩效面谈的目的不是盖棺论定，而是开启未来。

绩效面谈 = 营造正式感 + 换框架

  • 营造正式感：时间、空间、话语
  • 换框架：让员工在更加全局的视角看待自己的工作

换框架示例：

  1. 你知道明年公司最重要的目标是什么吗？
  2. 你知道公司的目标对咱们部门意味着什么吗？
  3. 你知道这对你意味着什么吗？

对绩效结果评价需要非常标准和清晰，且针对性行动也需要足够清晰。

主持会议 = 行为设计 + 全程控场

行为设计：

  1. 设计场地
  2. 设计时间
  3. 设计环境
  4. 设计规则

全程控场：（主持）

  1. 会前积极跑动
  2. 开场自我赋能（简洁、直白，不宜太刷存在感）
  3. 会中敢于干预
  4. 结束出口成章（把每个人的意见都纳入进来）

会议纪要的核心要素是决议

会前寒暄：请教周围同事最近发生的事情或问题。

道歉 = 关闭过去 + 承诺未来

关闭过去：

  1. 道歉越早，代价越小
  2. 分开处理道歉和弥补损失
  3. 态度诚恳
  4. 接纳对方的情绪

承诺未来：

  1. 带着新方案
  2. 上个请教
  3. 上个小礼物

道歉的终极奥义不是承认我错了，而是承认你是对的

调解矛盾 = 处理情绪 + 重建目标 + 最小改善

处理情绪：接纳对方情绪
重建目标：清晰冲突者的目标
最小改善：设定一个和目标相关最小化、可立即执行的动作，且不和矛盾相关。

求助 = 别人帮得了 + 自己值得帮

别人帮得了：

  1. 时间和精力上可以启动
  2. 在职责边界内科启动
  3. 在关系程度内可启动

自己值得帮：

  1. 先干为敬
  2. 真诚领情

求助的本质是发起一次协作

拒绝 = 开门见山 + 移花接木

开门见山：直接拒绝，找到真实存在的理由。
移花接木：表明态度，给出替代方案

即兴发言 = 万能开头 + 谈感受 / 谈行动

万能开头：不贴负面标签，不增加冗余信息
谈感受 / 谈行动：素材来源于现场，烘托氛围，给别人捧场。

即兴发言不是毫无准备的讲话，它不能准备稿子，但一定要“准备意识”

闲谈 = 传递尊重 + 传递价值

传递尊重：

  1. 上个请教 - 对方擅长的领域，明确的场景性问题、粒度不宜太大
  2. 多给一点 - 提出开放性问题
  3. 深度破冰 - 寻找与这些人有特殊关系的事件和故事，可以强准备
  4. 管理人设 - 聊一些符合自己人设的话题，不符合人设的话题，即时懂也少聊。注意边界，三观类话题不适合闲谈。

传递价值：让对方感受好，有收获感。

提案 = 融目标 + 抢进度 + 提诉求

融目标：将提案的目标和客户的目标融合起来
抢进度：增强说服力和责任担当

  • 抢影响实质性结果的工作（帮助决策的工作）
  • 把展现自己决心的工作往前做

竞聘不是为了输赢，而是表现未来长远的信心

我们可以不看重当时当下那场“有限游戏”的输赢，但要努力争取在无限游戏中向前迈进一大步。

竞聘 = 请战

  1. 肯定现状
  2. 详述接下来的具体事项

竞聘 = 肯定现有战场 + 我的独特打发 + 我的充分准备

注意事项：

  1. 不要说前任的坏处，要肯定现有战场
  2. 不要摆过去的功劳，要讲未来的打算
  3. 不要说对未来的畅想，要说对未来的抓手
  4. 不要光讲准备，还要突出个人特质
  5. 不要因为落选闹情绪，要备好败选方案
